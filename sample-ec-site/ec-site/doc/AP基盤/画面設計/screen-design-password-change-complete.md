# パスワード変更完了画面 設計書

## 1. 概要

- **画面名**: パスワード変更完了画面
- **URL**: `/account/password/change/complete` (想定)
- **ファイルパス**: `presentation/src/main/resources/templates/account/passwordChangeComplete.html` (想定)
- **概要**: パスワードの変更が正常に完了したことをユーザーに通知し、メニュー画面へ遷移させるための中間画面。

## 2. 画面レイアウト

画面中央に完了メッセージと次に進むためのボタンを配置します。

```
+------------------------------------------------+
|                                                |
|         ECサイト - パスワード変更完了          |
|                                                |
+------------------------------------------------+
|                                                |
|  パスワードの変更が完了しました。              |
|                                                |
|                                                |
|             [ メニュー画面へ ]                 |
|                                                |
+------------------------------------------------+
```

## 3. 画面項目一覧

この画面にはユーザーが入力する項目はありません。

## 4. アクション

| No. | アクション名   | 概要                         | 遷移先             |
|:----|:---------------|:-----------------------------|:-------------------|
| 1   | メニュー画面へ | ボタンクリックでメニュー画面へ遷移する | メニュー画面 (`/menu`) |

## 5. 補足

- 本画面は `screen-flow.puml` の変更に伴い追加された画面です。
- 従来はパスワード変更後、直接メニュー画面へ遷移していましたが、完了を通知する専用画面を挟む構成に変更されました。
